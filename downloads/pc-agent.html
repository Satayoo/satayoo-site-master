Installation Instructions

  1. Download the installer above
  2. Run the installer as Administrator
  3. Follow the setup wizard
  4. Choose auto-start option (recommended)
  5. Agent will connect to api.satayoo.com automatically
  6. Approve voice commands when prompted

System Requirements

  • Windows 10 or Windows 11
  • Internet connection for cloud API
  • Chrome or Edge browser (for ChatGPT automation)
  • Administrator privileges for installation
  • 5 MB free disk space

PC Agent Features

Standalone Executable

Native C implementation with no Python dependencies. Fast, efficient, professional.

Mouse Automation

Direct ChatGPT.com control with intelligent keyboard navigation and agent mode activation.

Cloud Integration

Connects to api.satayoo.com to receive voice commands from your Android app.

User Approval

Security dialogs for every command. You control what gets executed on your computer.

Intelligent Routing

Automatically opens the right tool: ChatGPT for general tasks, Cursor for coding.

System Control

Handle screenshots, volume control, desktop management, and window operations.
